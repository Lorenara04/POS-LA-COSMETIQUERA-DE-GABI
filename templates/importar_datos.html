Instrucciones de Importación

Para importar correctamente su inventario, debe seguir estas reglas:

  1. El archivo debe ser formato **.xlsx** (Excel).
  2. El archivo debe tener una hoja llamada exactamente: "Producto".
  3. La importación **borrará todo el inventario existente** y lo reemplazará con los datos del Excel. ¡Úselo con cuidado!
  4. Las columnas son sensibles a mayúsculas y minúsculas y deben ser las siguientes (el orden no importa):

Columnas Requeridas/Opcionales:

Nombre Columna Obligatorio Tipo de Dato Descripción
nombre SÍ Texto Nombre del producto (e.g., Mascara Pestañas X).
valor_venta SÍ Número (Decimal) Precio final al público.
cantidad NO Número (Entero) Stock actual. Si falta, se asume 0.
codigo NO Texto/Número Código de barras o SKU único. Si falta, se genera automáticamente.
marca NO Texto Marca o fabricante.
descripcion NO Texto Descripción corta.
valor_interno NO Número (Decimal) Costo interno del producto.
